Instructions

  • Use Style Controls to format the table to be as easily readable as possible on a mobile phone screen.
  • Set the browser to 100% zoom. The Table On Mobile Preview area should be approximately the size of an iPhone 12.
  • Table On Mobile Preview area always stays the same size. If the table does not fit, the area will be scrollable.
  • To preview the table on your own mobile device, scan the QR code.
  • To preview the table in another browser tab or window, click the Preview button.
  • When done, add your User Id (no spaces) and click Submit.
